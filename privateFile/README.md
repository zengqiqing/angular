# 接口使用说明!
## 即将上映
### 获取所有即将上映电影数据!
- 请求地址: http://localhost:3000/coming_soon
- 请求类型: GET
- 请求参数: 无

  | 参数名  | 数据类型 | 参数说明 |
  | ---- | ---- | ---- |
  |      |      |      |

### 分页获取即将上映电影数据
- 请求地址: http://localhost:3000/coming_soon

- 请求类型: GET

- 请求参数: 

  | 参数名    | 数据类型   | 参数说明        |
  | ------ | ------ | ----------- |
  | _page  | Number | 指定要获取第几页的数据 |
  | _limit | Number | 指定要获取几条数据   |
